# Your Input under one minute

## 1. Build a controlled Input.

```javascript
import { Input } from 'react-oxygen-ui';

const GeneralInput = () => {
  const [text, onTextChange] = useState('');

  return (
    <Input
      type="text"
      value={text}
      onChange={e => {
        onTextChange(e.currentTarget.value);
      }}
    />
  );
};
```

## 2. Add some style

```javascript
import { Input } from 'react-oxygen-ui';

const GeneralInput = () => {
  const [text, onTextChange] = useState('');

  return (
    <Input
      type="text"
      padding="8px"
      border="1px solid"
      borderColor="#dfe4ea"
      borderRadius="4px"
      value={text}
      onChange={e => {
        onTextChange(e.currentTarget.value);
      }}
    />
  );
};
```

## 3. Add validation feature

It will receive 2 props:

1. `validate`: a regexp for validate the user input
2. `errorMsg`: a string for error message when validation failed

- When error, the border color should change to red.<br/>
- Error message is red, and success message will be green.

```javascript
import { Input } from 'react-oxygen-ui';

const GeneralInput = ({ validate, errorText }) => {
  const [input, onInputChange] = useState('');
  const isCorrectInput = input && validate(text);

  return (
    <Box direction="column" gap="8px">
      <Input
        type="text"
        padding="8px"
        border={isCorrectInput ? '1px solid' : '2px dashed'}
        borderColor={isCorrectInput ? '#dfe4ea' : 'red'}
        borderRadius="4px"
        value={input}
        onChange={e => {
          onInputChange(e.currentTarget.value);
        }}
      />
      <Text fontSize="12px" color={isCorrectInput ? 'green' : 'red'}>
        {isCorrectInput ? 'Great, you get it right' : errorText}
      </Text>
    </Box>
  );
};
```

Notice the [gap](https://react-oxygen-ui.albertgao.now.sh/docs-2-components-2-1-box) prop? It's a handy prop from `<Box>`, it will add gap between each children, no need to worry about the margin any more.

## 4. Add onHover style

```javascript
<Input
  // remove the other props for simplicity
  css={{
    '&:hover': {
      borderColor: 'blue',
    },
  }}
/>
```

## 5. Now, can we do better?

Everything works fine. With this example, you should have a idea of how `react-oxygen-ui` works. You can use all the other components like this `<Input>`.

And you can start building your own components library now.

But we can do better, all these `8px`, `1px solid`, `#dfe4ea`, they are magic numbers, if you are building a production grade application, you might want to depend on a design system.
