## Properties

All `props` from `<BaseBox>`. With the following extra.

```typescript
interface IInput extends IBaseBoxWithTakeRef {
  type?: InputTypes;
  disabled?: boolean;
  width?: Responsive;
  height?: Responsive;
  value?: string | number;
  placeholder?: string | number;
  name?: string;

  onChange?: React.ReactEventHandler;
  onBlur?: React.ReactEventHandler;
}
```
